# AI Learning Platform - Prompt Management

## 🚀 Quick Start
1. **Clone the repository**
2. **Setup Environment Variables:** Create a `.env` file in the `backend` folder.
3. **Run Database:** `docker-compose up -d`
4. **Run Backend:** `cd backend && npm install && npx tsx src/app.ts`

## 🛠 Tech Stack
- **Backend:** Node.js, Express, TypeScript
- **Database:** MongoDB (Running in Docker)
- **Validation:** Mongoose

## 📡 API Endpoints (Work in Progress)
- `POST /api/users/register` - Register a new user
